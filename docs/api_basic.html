Synthesis Basics#

The top-level module letalker directly houses all the commonly used features of pyLeTalker that are essential to run the voice synthesis while more advanced features remain only in their respective submodules. The following lists summarize all the top-level functions and classes.

  • Sampling rate and time vector

    fs

    int([x]) -> integer int(x, base=10) -> integer

    ts

    Return time vector in seconds.

  • Simulation Functions

    sim

    Run simulation

    sim_kinematic

    Run simulation with kinematic vocal fold model

  • Synthesis Elements

    Vocal Folds classes

    LeTalkerVocalFolds

    Self-oscillating 3-mass model (Titze and Story, 2002)

    KinematicVocalFolds

    Kinematic vocal fold model (Titze 1984)

    VocalFoldsUg

    Vocal source model with known glottal flow function

    VocalFoldsAg

    Vocal fold model with known glottal area waveform

    Vocal Tract classes

    LeTalkerVocalTract

    Wave-reflection vocal tract model (Liljencrants, 1985; Story, 1995)

    LossyCylinderVocalTract

    Lossy Straight Tube Vocal Tract model

    Lungs class

    LeTalkerLungs

    Constant-pressure lung model with impedance matching

    OpenLungs

    Constant-pressure lung model with zero-reflection

    NullLungs

    Zero-pressure lung model, only reflecting 80% of the backward pressure

    Lips class

    LeTalkerLips

    Ishizaka-Flanagan

  • Function Generators

    There are three types of generators to produce parameter samples.

    • Standalone Generators

      Constant

      Constant function generator

      StepGenerator

      Step function generator

      LineGenerator

      Line generator

      Interpolator

      PeriodicInterpolator

      Periodic interpolator

      ClampedInterpolator

      Clamped interpolator

      FlutterGenerator

      Generates quasi-random sum of sinusoids (generalization of Klatt & Klatt, 1990)

      SineGenerator

      Sine wave generator (analytic)

      ModulatedSineGenerator

      Modulated sine wave generator (analytic)

      RosenbergGenerator

      Rosenberg's pulse trains (Rosenberg, 1971)

    • Function Modifiers

      ProductGenerator

      Combines function generator outputs by multiplication

      SumGenerator

      Combine function generator outputs by summation

      ExponentialGenerator

      Generates exponential function generator wrapper

      LogGenerator

      Log function generator

    • Random Noise Generators

      WhiteNoiseGenerator

      White noise generator

      ColoredNoiseGenerator

      Colored noise generator

Simulation Functions#

letalker.sim(nb_samples, vocalfolds, vocaltract, trachea=None, lungs=None, lips=None, aspiration_noise=None, *, return_results=True, n0=0)#

Run simulation

Parameters#

nb_samples

Number of samples to generate

vocalfolds

Vocal fold model

vocaltract

Supraglottal vocal tract. If a 2-letter vowel name (e.g., 'aa' & 'ii') is passed, the default LeTalkerVocalTract model will be used with a male vocal tract, configured to the specified vowel. If 1D or 2D array or a FunctionGenerator object is passed, it will be used as the cross-sectional areas (in cm) of vocal tract sections (if 2D, the first dimension is the time axes). To specify other parameters of the LeTalkerVocalTract, you may pass in a dict of key-value pairs. A custom VocalTract model object could also be passed.

trachea, optional

Subglottal vocal tract model, by default None to use the LeTalkerVocalTract model with the default trachea cross-sectional areas ('trach'). If 1D or 2D array or a FunctionGenerator object is passed, it will be used as the cross-sectional areas (in cm) of vocal tract sections (if 2D, the first dimension is the time axes). To specify other parameters of the LeTalkerVocalTract, you may pass in a dict of key-value pairs. A custom VocalTract model object could also be passed.

lungs, optional

Lungs model, by default None to use LeTalkerLungs model with the default 7840 dyn/cm lung pressure level (with the onset time of 2 ms). A float value changes the pressure level while a FunctionGenerator object can be specified to alter the temporal profile. A custom Lungs model object could also be passed.

lips, optional

Lips model, by default None to use LeTalkerLips model (Ishizaka-Flanagan lips radiation model). A custom Lips model object could also be passed.

aspiration_noise, optional

Aspiration noise model, by default None to disable aspiration noise injection (or keep the default aspiration noise mode of the specified VocalFolds model). Passing in True enables the injection model (LeTalkerAspirationNoise) at the default level and threshold. Use a key-value dict to customize LeTalkerAspirationNoise further or pass in a custom AspirationNoise model object.

return_results, optional

If True, also return the results dict of all the simulation elements, by default True

n0, optional

Simulation starting time in sample index, by default 0

Synthesis Elements#

There are 4 voice production element types:

  • Vocal Folds (Glottis)

  • Vocal Tract (for both subglottal and supraglottal elements)

  • Lungs

  • Lips

pyleTalker provides at least one built-in class for each type.

Also, an aspiration noise model can be assigned to a vocal folds class to inject turbulent aspiration noise which is a function of the glottal flow.

Parameters:

  • x0 (SampleGenerator) prephonatory displacement (lower, upper)

  • Lo (float) vocal fold length at rest

  • To (float) vocal fold thickness at rest

  • zeta (NDArray) damping ratios [lower, upper, body], defaults to np.array([0.1, 0.6, 0.1])

  • upstream (VocalTract | float | None) connected subglottal tract, defaults to None

  • downstream (VocalTract | float | None) connected supraglottal tract, defaults to None

  • aspiration_noise (bool | dict | AspirationNoise | None) True to use aspiration noise model, defaults to None
